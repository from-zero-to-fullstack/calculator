# Setup

## Run locally

Create and go to the directory where you want to place the repository

```bash
  cd my-directory
```

Clone the project

```bash
  git clone https://github.com/from-zero-to-fullstack/calculator.git
```

Go to the project directory

```bash
  cd calculator
```

Open in Visual Studio Code

```bash
  code .
```
